# Guandan Calculator - Project Overview

## Purpose
A comprehensive web-based scoring calculator for Guandan (掼蛋), a popular Chinese climbing card game. The application features:
- Real-time multiplayer room sharing with 6-digit room codes
- Team-based level progression (2→A) with 4/6/8 player modes
- 14 data-driven honor calculations with cultural Chinese gaming references
- Community voting system for MVP/burden selection
- Mobile-optimized exports (PNG, TXT, CSV)

## Target Users
- Chinese card game players who need accurate scoring and level tracking
- Groups playing together (offline gatherings or remote gaming via video call)
- Tournament organizers needing official scoring with spectator access
- Families playing casual games with quick setup

## Key Features
1. **Real-Time Room Sharing**: Create rooms with 6-digit codes, host controls game while viewers watch in real-time
2. **Smart Player Management**: 77+ emoji avatars, bulk name input, drag-drop team assignment
3. **Game Scoring**: Support for 4/6/8 player modes with automatic upgrade calculations
4. **Honor System**: 14 sophisticated algorithms for player achievements (吕布, 阿斗, 石佛, etc.)
5. **Community Voting**: Anonymous viewer voting for MVP/burden with live counting
6. **Export System**: Real-time room codes, mobile/desktop PNG, TXT/CSV formats
